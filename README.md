# 🌱 Hydroponie Automatisée

## 📌 Description
Ce projet consiste en un système d’hydroponie automatisé permettant de faire circuler de l’eau enrichie en nutriments pour alimenter des plantes sans sol.

Un moteur placé à la base du système assure la circulation continue de l’eau dans un circuit fermé, garantissant une irrigation efficace et contrôlée.

---

## ⚙️ Fonctionnement

Le système fonctionne selon le principe suivant :

1. L’eau est stockée dans un réservoir situé en bas  
2. Une pompe (moteur) envoie l’eau vers le haut  
3. L’eau circule à travers les plantes  
4. L’eau redescend par gravité vers le réservoir  
5. Le cycle recommence en continu  

### ✅ Avantages
- Économie d’eau  
- Croissance plus rapide des plantes  
- Système autonome  

---

## 🧩 Composants utilisés

- 🔌 Pompe à eau (moteur)  
- 🪴 Structure verticale (support des plantes)  
- 💧 Réservoir d’eau  
- 🔄 Circuit de tuyaux  
- ⚡ Alimentation (12V ou autre selon pompe)  
- 🧠 (Optionnel) Arduino pour automatisation
